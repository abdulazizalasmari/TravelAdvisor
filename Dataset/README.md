# Dataset Folder

## Purpose
This folder contains the raw travel-related dataset(s) used for the Travel Destination Advisor System.

## Dataset Sources
You can find suitable travel datasets from:
- **Kaggle:** https://www.kaggle.com/datasets
  - Search for: "travel destinations", "tourism", "trip advisor reviews", "travel recommendations"
- **GitHub:** Travel-related datasets repositories
- **Hugging Face Datasets:** https://huggingface.co/datasets

## Recommended Datasets
Some suggested travel-related datasets:
1. **Trip Advisor Reviews** - Reviews and ratings for destinations/hotels
2. **Travel Package Dataset** - Destination features, costs, and customer preferences
3. **Tourist Destination Ratings** - Destination characteristics and visitor ratings
4. **Flight and Hotel Booking Data** - Booking patterns and preferences

## Instructions
1. Download a suitable dataset (CSV, JSON, or similar format)
2. Place the raw data file(s) in this folder
3. Do NOT modify the raw data here - all preprocessing should be done in the notebooks
4. Update this README with the actual dataset name and source once selected
